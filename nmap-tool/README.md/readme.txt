# Basic Nmap Network Tool

A Python-based network scanning tool that retrieves detailed information about devices and scans the network for open ports.

## Features
- Scan specific devices for detailed information (OS, MAC address, uptime, ports).
- Scan the network for open ports across multiple devices.

## Prerequisites
- Python 3.x
- `nmap` installed on your system.
- `python-nmap` library (`pip install python-nmap`).

## Usage
Run the script:
```bash
python nmap_tool.py
